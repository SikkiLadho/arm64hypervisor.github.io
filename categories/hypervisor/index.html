Tutorial 1: Exception Levels

Today, we try to write a simple bootable hypervisor binary with just a blank screen(have patience, we’ll make it print the hello world too, but in future). Exceptions levels define execution privelege at each level. There are 4 exception levels in ARMv8 architecture. Exception Level 0 (EL0): This Exception Level or Privelege Level is commonly used by userspace applications. Firefox, Libreoffice, VLC Media Player, VirtualBox are few examples of userspace applications....
